围绕NodeJS搭建服务器,实现前后端交互开展接下来的课程

一前后端概览
请求 -> 响应 -> 数据解析 -> 页面渲染
1.前端发起请求(请求方式,请求头,请求体,接受到响应之后再进行数据处理,页面渲染)
2.后端搭建服务器(存储数据,存储文件),建库建表(构思整个项目中所需要存储的数据,并考虑响应式的索引及表关联等),
编写接口(指定请求地址对应的功能和业务,例:/login对应登录功能);与前端联调接口:接收到前端发来的请求,
执行对应接口中定义的方法,链接数据库,进行表记录操作后将操作结果响应回前端

二代码实现
1.搭建web服务器,编写简单接口 -> 启动数据库服务器.创建数据库,数据表结合接口实现对表记录增删改查
2.编写前端页面,并向服务器发起指定接口的请求 -> 对响应数据进行处理

三借助nodejs搭建web服务器
1.安装nodejs(在官网下载,不要最新,尽量求稳),会附带安装npm(node pacage manager.node 包管理工具
帮助我们安装,下载,更新node的依赖包).
2.不管什么语言,哪一个框架,真正项目中都会有集成好的快捷工具(脚手架工具CLI,例如:node-脚手架工具CLI,vue-脚手架)
等创建标准化的项目结构
